# ReceiptTexts

Export your iMessage conversations to PDF — simple, fast, and 100% private.

## Features

- Export full iMessage conversations to PDF
- Stays entirely on your device — no accounts, no servers, no data collection
- Fast and simple with no subscriptions or ads
- Supports iPhone, iPad, and Mac

## Requirements

- iOS 26.2+ / macOS 26.2+
- Xcode 26.3+

## Getting Started

1. Clone the repo
   ```bash
   git clone git@github.com:rudispallacci-bit/ReceiptTexts.git
   ```
2. Open `ReceiptTexts.xcodeproj` in Xcode
3. Select your target device and run

## Running Tests

```bash
xcodebuild test \
  -project ReceiptTexts.xcodeproj \
  -scheme ReceiptTexts \
  -destination 'platform=macOS'
```
